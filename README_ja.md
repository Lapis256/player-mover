# player-mover
[English](./README.md) / [日本語](./README_ja.md)

プレイヤーの座標を編集するMinecraftのツールです。

## 使い方
```
player_mover <WORLD> [CHANGE_POSITION] [CHANGE_DIMENSION]
```
- WORLD (必須)
  - ワールドディレクトリ
- CHANGE_POSITION (任意)
  - 指定した座標にプレイヤーを移動させます。
  - 形式: x,y,z
  - 指定していない場合はスポーン座標に移動させます
- CHANGE_DIMENSION (任意)
  - 指定したディメンションにプレイヤーを移動させます
  - 指定していない場合はオーバーワールドに移動させます
